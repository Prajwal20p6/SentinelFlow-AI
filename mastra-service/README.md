# SentinelFlow AI — Mastra AI Agent Microservice

This is the official Node.js microservice running Mastra to orchestrate AI agent workflows (RCA, threat intelligence, prioritization, and remediation planning).

## Setup & Running

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run in Development (Hot-reloading):**
   ```bash
   npm run dev
   ```

3. **Build TypeScript:**
   ```bash
   npm run build
   ```

4. **Start Production Build:**
   ```bash
   npm run start
   ```
