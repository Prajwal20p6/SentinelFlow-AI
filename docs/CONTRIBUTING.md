# Contributing Guide

Thank you for contributing to SentinelFlow AI!

## Code Contributions

### 1. Fork and Clone
Fork the repository and clone your fork locally:
```bash
git clone https://github.com/your-username/sentinelflow-ai.git
```

### 2. Set Up Environment
Follow the setup instructions in [WINDOWS-SETUP.md](../WINDOWS-SETUP.md) or run `setup.ps1` to prepare python venv and node packages.

### 3. Implement Changes
- Write tests for any new functionality.
- Ensure format checks pass.
- Write clear comments for complex logic blocks.

### 4. Create Pull Request
Submit a pull request to the `dev` or `main` branches of the main repository. Provide details on what changes were introduced and how they were tested.
